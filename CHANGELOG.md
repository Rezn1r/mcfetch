## [1.1.1] - 2026-01-7

- Added `--version` flag to display version information.
- Added `--update` flag to update mcfetch to the latest release.
- Added `--uninstall` flag to uninstall mcfetch from the system.

### Changed
- Updated README.md to include new flags and usage examples.

- Fixed minor bugs in the update and uninstall processes.
- Fixed installer
